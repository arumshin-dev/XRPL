🚀 XRPL Express API

XRPL 핵심 기능을 위한 RESTful API 서버

서버 상태: 확인 중...

💼 Wallet

지갑 생성, 로드, 펀딩 및 정보 조회

  • POST /api/wallet/create
  • POST /api/wallet/load
  • POST /api/wallet/fund
  • GET /api/wallet/info/:address

💸 Payment

XRP 및 IOU 토큰 전송

  • POST /api/payment/xrp
  • POST /api/payment/iou

🤝 TrustSet

신뢰선 생성 및 승인

  • POST /api/trustset/create
  • POST /api/trustset/authorize

⚙️ AccountSet

계정 플래그 및 설정 관리

  • POST /api/accountset/flags

🎫 Credential

신원 증명 발급, 수락, 조회, 삭제

  • POST /api/credential/create
  • POST /api/credential/accept
  • GET /api/credential/check/:address
  • POST /api/credential/delete

📈 Permissioned DEX

권한 기반 탈중앙화 거래소

  • POST /api/permissioned-dex/offer/create
  • GET /api/permissioned-dex/book
  • POST /api/permissioned-dex/offer/cancel

🏛️ Permissioned Domains

도메인 기반 접근 권한 관리

  • POST /api/permissioned-domains/create
  • POST /api/permissioned-domains/delete
  • GET /api/permissioned-domains/info/:domainId

🔒 Token Escrow

IOU 및 MPT 에스크로 관리

  • POST /api/token-escrow/create/iou
  • POST /api/token-escrow/create/mpt
  • POST /api/token-escrow/finish
  • POST /api/token-escrow/cancel

🪙 MPT (Multi-Purpose Token)

MPT 발행, 승인, 전송, 삭제

  • POST /api/mpt/issuance/create
  • POST /api/mpt/authorize
  • POST /api/mpt/send
  • POST /api/mpt/issuance/destroy

📦 Batch

배치 트랜잭션 처리

  • POST /api/batch/all-or-nothing
  • POST /api/batch/independent
  • POST /api/batch/only-one
  • POST /api/batch/until-failure

🌊 AMM

자동화된 마켓 메이커

  • POST /api/amm/create
  • GET /api/amm/info
  • POST /api/amm/deposit
  • POST /api/amm/withdraw
  • POST /api/amm/vote

🖥️ Server

서버 정보 및 상태 확인

  • GET /api/server/info
